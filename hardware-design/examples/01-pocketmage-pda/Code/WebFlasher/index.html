Installation Options

When "Preserve user data" is checked, only application partition will be flashed, keeping your settings and saved content.

When unchecked, a complete installation will be performed, which erases all data and settings.

Note: SD card data is NOT affected by this option.

Requirements

This tool requires a Chromium-based browser (Chrome, Edge, Opera) with Web Serial API support.

Connect your device via USB and ensure no other program is using the serial port.
